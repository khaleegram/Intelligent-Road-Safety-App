# Redlines

## Spacing
- Screen horizontal padding: `space.4` on mobile.
- Vertical block rhythm: `space.4`.
- In-card content spacing: `space.2` to `space.3`.

## Typography
- Section labels: `font.size.xs`, `font.weight.bold`.
- Body content: `font.size.sm`, `font.weight.regular`.
- Page titles in top bar: `font.size.md` to `font.size.lg`.

## Control Sizes
- Touch target minimum: `44x44`.
- Dense table/list rows: `48` to `56` minimum row height.

## Top Bar
- Capsule radius: `radius.xl`.
- Surface: `color.surfaceElevated`.
- Elevation: `shadow.md`.
- Collapse threshold: `space.2`.
